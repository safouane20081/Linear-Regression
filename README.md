# 📈 Linear Regression Projects

A collection of Python machine learning projects exploring different types of linear regression models.

## 📂 Projects

### 1. Single Variable
Predicting output based on one input feature.

### 2. Multiple Variables
Predicting output based on multiple input features.

### 3. Dummy Variables
Predicting Manhattan real estate prices using neighborhood names encoded as dummy variables.

## 🛠 Requirements
- Python 3.x
- pandas
- numpy
- scikit-learn

## ▶️ How to run
```bash
pip install pandas numpy scikit-learn
python main.py
```
